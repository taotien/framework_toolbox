# Framework Toolbox

Quick and dirty GUI for utilities of the Framework Laptop

## Installation

Note: **I am new to rust, please look through the code and taste the spaghetti
before installation**

```sh
cargo install --git "https://github.com/taotien/framework_toolbox.git"
```

## Dependencies

- rust
- ectool
- iio-sensor-proxy
- Polkit/pkexec
